  • In this lesson, we will work on a new problem and learn about general problem solving techniques.

Problem...

"Given a day, the program should print if it is a weekday or weekend."

Example...

    $ python3 day_of_the_week.py monday
    weekday

    $ python3 day_of_the_week.py sunday
    weekend

Techniques...

  • Divide the problem into smaller ones.

  • Write something that works and slowly improve it.

  • Solve the problem for some example inputs and then write the program.

Home work...

Given a word, your program should print number of vowels in the word.

Examples...

    $ python3 count_vowels.py "test"
    Number of vowels: 1

    $ python3 count_vowels.py "it is snowing"
    Number of vowels: 4

    $ python3 count_vowels.py "xyz"
    Number of vowels: 0
